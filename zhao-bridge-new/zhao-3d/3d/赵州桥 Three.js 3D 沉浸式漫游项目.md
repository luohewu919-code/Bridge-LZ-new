# 赵州桥3D模型交互展示项目
## 项目简介
基于 Three.js 实现的赵州桥 OBJ/MTL 3D模型在线交互演示，包含：
- 3D模型加载与材质渲染
- WASD 键盘自由漫游
- 鼠标拖拽视角旋转
- 8个建筑知识点点击弹窗介绍（左图右文布局）
- 3D星空粒子背景特效
- 标记点光环呼吸动画
- 窗口自适应、射线拾取交互

## 功能特性
1. **模型渲染**
- 支持 OBJ + MTL 材质模型加载
- 双面材质渲染，防止模型背面发黑
- 环境光+平行光组合光照

2. **操控方式**
- WASD：前后左右移动相机
- 鼠标按住拖拽：旋转视角
- 视角俯仰角度限制，防止穿模

3. **交互点位**
内置8个建筑科普点位：
- 蚣蝮（吸水兽）
- 敞肩小拱
- 主拱结构
- 铁件榫卯
- 桥头石狮
- 望柱雕饰
- 龙兽浮雕
- 蛟龙穿岩栏板

4. **视觉特效**
- 全屏3D星空粒子缓慢自转
- 点位外圈光环呼吸缩放+透明度动画
- 深色场景背景，粒子层次感更强
- 点击弹出黑色半透明信息弹窗

## 运行方式
1. 将所有文件放在同一个文件夹
2. 用本地服务器打开（Live Server / Nginx / VSCode 插件）
3. 禁止直接双击打开（跨域无法加载模型与图片）

## 可自定义修改参数
### 粒子特效
- starCount：粒子数量
- size：粒子大小
- color：粒子颜色
- 旋转速度：animate 中 `starParticles.rotation` 数值

### 操控灵敏度
- moveSpeed：相机移动速度
- mouseSensitivity：鼠标视角灵敏度
- pitchLimit：视角上下旋转最大角度

### 样式
- #floatTip：弹窗宽高、背景色、圆角、字体
- 标记点球体大小、光环大小、呼吸动画频率

## 技术栈
- HTML5 + CSS3
- JavaScript ES6 Module
- Three.js r160
- OBJLoader / MTLLoader 模型加载器
- 射线投射 Raycaster 点击拾取
- BufferGeometry 3D粒子系统
